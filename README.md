## duplicate_file_cleaner
このプログラムは、ファイル名の末尾に (1) や （1） といった連番が付いている重複コピーファイルを自動で検知し、一括削除するためのツールです。
Windowsなどでファイルをコピーしたり、同じファイルを複数回ダウンロードした際に生成される「コピーされた不要なファイル」を自動で検知し、一括で削除するためのツールです。

## 概要
このプログラムを特定のフォルダに配置して実行すると、ファイル名の末尾に (1) などの連番が付いているファイルをスキャンします。
実行前に確認メッセージが表示されるため、意図しない削除を防ぐ安全設計になっています。

## 特徴
Windows対応: Windowsでよく使われる半角 (1) と、全角 （1） の両方のパターンを検知します。
誤削除防止:
ファイル名の末尾が「カッコ＋数字」になっているものだけを正確に判別します（例：memo(temp).txtなどは削除されません）。
実行前にユーザーへ確認（y/n）を求めます。
スクリプト自身を削除対象から自動的に除外します。
インストール不要: Pythonさえ入っていれば、標準ライブラリのみで動作します。

## 使い方
1. 配置
削除したいファイルが溜まっているフォルダに delete_duplicates.py をコピーまたは移動します。

2. 実行
コマンドプロンプトやターミナルで以下のコマンドを実行します。

bash


python delete_duplicates.py
3. 確認
削除対象のファイルが見つかると、実行の確認が表示されます。
y を入力して Enter を押すと削除が実行されます。

## 削除対象のパターン例
ファイル名	判定
photo (1).jpg	✅ 削除対象 (半角)
文書（2）.pdf	✅ 削除対象 (全角)
data (10).csv	✅ 削除対象 (2桁以上もOK)
report.docx	❌ 対象外 (連番なし)
meeting(draft).txt	❌ 対象外 (数字ではない)

## 動作環境
Python 3.x
OS: Windows / macOS / Linux (Windowsのファイル命名規則を主な対象としています)
